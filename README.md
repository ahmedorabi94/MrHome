# MrHome
Android application to control and monitor the house

This android application will be used to monitor and control the house. It can help elderly or handicapped people live a more independent life as long as possible. This application will allow the user to control a device that is connected to any home appliance. The users are able to switch on/off the lights and switch on/off air condition system when you are ten away from home. The users could check the status of TV and door.
